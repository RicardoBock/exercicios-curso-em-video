Listas

Listas Ordenadas

Listas em que a ordem é de suma importância, como por exemplo uma receita, candidatos de um concurso, lista de valores, ex:

  1. Ricardo Bock Camargo
  2. Josef Climber
  3. Mark Russel
  4. Mark Wahlberg

Listas não Ordenadas

Forma de organizar algo em lista mas que não necessariamente necessite de uma classificação, como lista de compras, to-do lists e várias outras atividades, ex

  • Tomate
  • Banana
  • Tangerina

Possibilidade de listas

É possível incluir uma lista dentro da outra, como por exemplo

  1. Item 1
  2. Item 2
    1. item 2.1
    2. item 2.2
  3. Item 3
  4. Item 4

CSS
Linguagem de marcação para a especificação de estilos em sites
